# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2025-11-21

### Added
- Enhanced compatibility with modern browsers using Interop 2025 features
- Improved TypeScript typings for better developer experience

### Fixed
- React integration issues with attribute binding and event handling
- Communication problems between Web Components and frameworks
- Event propagation inconsistencies in framework environments

### Improved
- Performance optimizations for Houdini Worklets implementation
- Responsive design implementation with Container Queries
- Framework interoperability (React, Vue, Angular, Svelte)

## [0.1.0] - 2025-11-10

### Added
- Initial release of fluent-grow
- Zero-runtime CSS-in-TS framework leveraging Web Components and Houdini worklets
- Atomic design system with Quarks, Atoms, Molecules, Organisms, Templates, and Systems
- Support for modern browser features with automatic fallbacks
- Theme management with light/dark modes
- Comprehensive documentation and examples
